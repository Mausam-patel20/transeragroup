🚛 Transera Group Website
📌 Overview

Transera Group is a logistics and transportation company providing reliable freight and shipping solutions across Canada and North America. The platform showcases services, company information, and allows users to connect for logistics solutions.

The website is designed to highlight efficient, secure, and timely transportation services, helping businesses streamline their supply chain operations.

📦 Services Offered

Transera provides a range of logistics services, including:

Full Truckload (FTL) – Dedicated capacity for large shipments
Less Than Truckload (LTL) – Cost-effective shipping for smaller loads
Trans-Border Shipping – Canada–USA freight solutions
Refrigerated Transport (Reefer) – Temperature-controlled shipments
Dry Van Services – General freight transportation
Expedited Shipping – Fast delivery for urgent shipments

⚙️ Installation & Setup
# Clone the repository
git clone https://github.com/your-username/transera-group.git

# Navigate into the project
cd transera-group

# Install dependencies
npm install

# Run development server
npm run dev
🚀 Deployment

This project is deployed using Render.

To deploy:

Push code to GitHub
Connect repository to Render

Set build command:

npm install && npm run build

Set start command:

npm start
